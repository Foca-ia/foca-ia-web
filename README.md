## Passos para Executar Localmente
# Clone o repositorio

``` bash
git clone https://github.com/OsapiCare/osapi-care-web-app
cd osapi-care-web-app
```

# Instale as dependencias

```bash
npm i
# or
yarn
```

# Execute localmente

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
